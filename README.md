# SlideCraft V2

HTMLプレゼンテーションの編集・調整・PDF書き出しを、デザインを崩さずに行うためのツールです。

## 使い方（ローカル起動）

1. ターミナルで `d:\MyPC\OneDrive - sorailo\Python\BEtProject\html-presentation-editor` に移動します。
2. 以下のコマンドを実行します：
   ```bash
   npm run dev
   ```
3. 表示されたURL（通常は `http://localhost:5173`）をブラウザで開きます。

## 主要な操作
- **HTML貼り付け**: 右上のアイコンから、外部のHTMLコードをそのままスライドとしてインポートできます。
- **画像編集**: スライド内の画像をクリックすると編集バネルが表示されます。拡大・縮小や位置の微調整が可能です。
- **書き出し**: 
  - **HTML**: 右上のエクスポートボタン。閲覧用の「プレゼンテーションモード」の有無を選べます。
  - **PDF/印刷**: プリンタアイコン。A4横固定で出力されます。
- **スライド切り替え**: 左側のサムネイルをクリック、または非編集時に矢印キー。
