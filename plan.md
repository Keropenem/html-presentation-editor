# SlideCraft V2 - 開発状況と今後の予定

## 1. 完了した機能
- **Shadow DOM による隔離**: スライドのCSSがエディター本体に干渉しない堅牢な設計。
- **A4 ランドスケープ対応**: スライドサイズを 297mm x 210mm に固定し、印刷精度を向上。
- **画像クロッピング（決定版）**: `object-fit: cover` と `transform-origin` の同期により、歪みゼロ・全方向パン可能な調整を実現。
- **テキスト編集機能**: 見出しや段落の「太字・文字色・行間」を動的に変更可能。
- **プレゼンテーションモード付きHTMLエクスポート**: 書き出したHTML単体でプレゼンが可能。
- **デスクトップ起動**: デスクトップのバッチファイルからワンクリックで起動可能に。

## 2. 次回の課題
- **PDF書き出しの1ページ目ズレ修正**: 現在は安全のため旧バージョン（.print-pageのみ）を維持。余白計算の再調整が必要。
- **サムネイル**: サムネイルが適切に表示されない問題。
- **リッチテキスト編集**: 文字の色や太さ、斜体など一般的なテキスト編集機能の実装。

## 3. 起動方法
1. デスクトップの「SlideCraftを起動.bat」をダブルクリック
2. ブラウザが自動で開き、エディターが利用可能になります
